# 日期时间工具

为 AI 助手提供实时日期与时间查询能力。

## 使用场景

当用户提出以下问题时，应该调用此 skill：

- 现在几点？
- 今天几号？
- 今天星期几？
- 现在是什么日期时间？
- 帮我看一下当前时间
- 东京现在几点？
- 帮我查一下上海现在时间

## 工具定义

### 1. get_current_datetime

获取当前日期和时间。

**参数：**

- `timezone` (string, optional): IANA 时区名称，如 `Asia/Shanghai`、`Asia/Tokyo`、`UTC`

**返回格式：**

```json
{
  "success": true,
  "timezone": "Asia/Shanghai",
  "date": "2026-03-08",
  "time": "18:30:15",
  "datetime": "2026-03-08 18:30:15",
  "iso": "2026-03-08T18:30:15+08:00",
  "weekday": "Sunday",
  "weekday_cn": "星期日",
  "timestamp": 1772965815
}
```

## 回复建议

- 用户只问几点时，优先简洁回答时间
- 用户问日期时，优先回答日期和星期
- 用户指定时区时，明确带上时区名称
- 如果工具返回错误，直接说明时区无效或获取失败

## 注意事项

- 不要凭记忆回答当前时间
- 需要实时调用工具
- 如果用户没指定时区，默认使用服务器当前本地时区
